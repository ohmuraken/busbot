# busbot
バスの時刻表を教えてくれるSlackのbotです.

## 対応しているバスの系列
- 京都市バス北3系統 北大路バスターミナル行き (京産大発)
- 京都バス40系統 国際会館行き (京産大発)


## 使い方
- ```@busbot [系統],[時間]``` のように使います
- 北3系統は```n3```, 40系統は```40```として下さい
- ```[時間]```を指定しない場合には, 現在時刻で処理されます.
